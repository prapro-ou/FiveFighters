# 概要
「ShapeGrazer」は、敵の弾をギリギリで避けることで攻撃を行うローグライトシューティングゲームです。

# ゲームの始め方
GitHub Pagesによって作成したWebページにアクセスすることでプレイできます。

URL: https://prapro-ou.github.io/FiveFighters/ (テストビルド)

# 操作方法
## メニュー操作
- 移動: WASD or 十字キー
- 決定: Enterキー

## キャラクター操作
- 移動: WASD or 十字キー
- 低速移動: Shiftキー + 移動
- スペシャルスキル: Oキー
- シフトスキル: J, K, Lキー
- ショップでの決定: Enterキー
