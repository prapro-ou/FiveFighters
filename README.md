# 概要
「ShapeGrazer」は、敵の弾をギリギリで避けることで攻撃を行うローグライト・シューティングゲームです。

# ゲームの始め方
GitHub Pagesによって作成したWebページにアクセスすることでプレイできます。

URL: https://prapro-ou.github.io/FiveFighters/

# 操作方法
## メニュー操作
- 移動: WASD or 十字キー
- 決定: Enterキー

## キャラクター操作
- 移動: WASD or 十字キー
- 低速移動: Shiftキー + 移動
- スペシャルスキル: Oキー
- シフトスキル: J, K, Lキー
- ショップでの決定: Enterキー
- ゲーム終了: Escapeキー
